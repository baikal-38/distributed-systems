Примерный план такой (Распределенные вычислительные системы)
Лабы:
1. Берете свою программу (или пишите новую) , делайте к ней JSON-интерфейс, т.е. делаете из нее микросервис.
2. Делайте Веб-приложение, взаимодействующее с вашей программой по протоколу HTTP.
3. Создаем сервис Docker и/или контейнер Kubernetes. 
4. Синхронизируем работу сервисов при помощи RabbitMQ.
